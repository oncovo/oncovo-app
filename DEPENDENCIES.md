# Dependências do Projeto

## Dependências Principais (a serem instaladas)

### Expo (Opcional - para usar EAS Build e funcionalidades Expo)
```bash
npm install expo
npm install --save-dev @expo/config-plugins
# Para EAS Build (opcional)
npm install -g eas-cli
```

### Supabase
```bash
npm install @supabase/supabase-js
```

### Google Authentication
```bash
npm install @react-native-google-signin/google-signin
```

### Notificações Push
```bash
npm install @react-native-firebase/app @react-native-firebase/messaging
```

### Navegação
```bash
npm install @react-navigation/native @react-navigation/native-stack
npm install react-native-screens react-native-safe-area-context
```

### Gerenciamento de Estado (opcional, mas recomendado)
```bash
npm install @tanstack/react-query
# ou
npm install zustand
```

### Utilitários
```bash
npm install react-native-config  # Para variáveis de ambiente
npm install react-native-async-storage  # Para armazenamento local
npm install react-native-image-picker  # Para seleção de imagens
npm install react-native-fast-image  # Para otimização de imagens
```

### Desenvolvimento
```bash
npm install --save-dev @types/react-native-config
npm install --save-dev babel-plugin-module-resolver
```

## Instalação por Etapas

Seguiremos a abordagem incremental conforme solicitado. As dependências serão instaladas conforme necessário para cada funcionalidade.
